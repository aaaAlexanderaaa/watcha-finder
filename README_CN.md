# Watcha Finder

一个用于发现和评估 AI 产品的 Claude Code 技能，基于 [watcha.cn](https://watcha.cn) 平台 API。

## 功能介绍

Watcha Finder 帮助你找到真正适合需求的 AI 产品——而不仅仅是最热门的。它能够：

- **搜索 watcha.cn 的 1000+ 产品数据库**，使用智能回退策略（API 不支持模糊搜索）
- **阅读真实用户评测**，理解超越评分的社区情感
- **区分热度与质量**——评测数和回复数代表热度，不一定代表好用
- **交叉引用网络资源**，避免因社区活跃度低而误判优质产品
- **提供细致入微的推荐**，清晰说明权衡取舍

## 核心特性

- 多策略搜索（文本查询、分类筛选、中英文关键词）
- 深度分析，提取评测文本内容
- 评分可靠性评估（标记评测数过少的产品）
- 发现黑马产品（找到小众但高质量的工具）
- 中国大陆可用性筛选
- 平台/价格/功能对比

## 安装方法

这是一个 Claude Code 技能。使用方法：

1. 将 `watcha-finder/` 目录复制到你的 Claude Code 技能位置
2. 当你询问 AI 产品相关问题时，技能会自动可用

## 工作原理

该技能使用 watcha.cn API 执行以下流程：

1. **广泛搜索** — 尝试多种查询策略（因为搜索是精确匹配）
2. **筛选候选** — 基于相关性和数据质量选择 3-5 个产品
3. **深度调研** — 获取产品详情、评测（最多 20 条）和社区讨论
4. **网络搜索** — 用外部视角补充 watcha 数据
5. **综合分析** — 呈现带有细致质量评估的发现

## API 参考

该技能访问以下 watcha.cn 端点：

- `POST /api/v2/search/general` — 产品搜索，支持分类/标签筛选
- `GET /api/v2/products/{id}` — 产品详情
- `GET /api/v2/products/{id}/reviews` — 用户评测
- `GET /api/v2/products/{id}/posts` — 社区讨论

## 局限性

- 搜索 API 是精确匹配，非模糊搜索（技能通过多查询策略处理）
- 评测/回复数表示热度，非质量（技能明确考虑这一点）
- 评测数 <10 的评分不可靠（技能会标记）
- watcha 活跃度低的产品可能仍然很优秀（技能补充网络搜索）
